# Space Pirates
### Terminal Shooter Game


#### To Play Space Pirates:
* clone repo
* cd into the cloned repo 
* copy and paste the following code into your terminal
```javascript
make re; ./bullet
```



#### Tech Used to Make This:
 * C++
 * curses library
